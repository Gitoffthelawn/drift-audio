# Credits

Drift bundles audio recordings from several sources under open
licences. This file contains the full attribution for each. In-app
credits are also available under the Credits menu.

All recordings have been edited for use in Drift — segmented, loop-baked,
EQ'd, compressed, and loudness-normalised. This processing is noted
here to comply with Creative Commons change-disclosure requirements.

Some recordings were sourced via the
[Muges/ambientsounds](https://github.com/Muges/ambientsounds) collection
(which packages them under CC0), but the original recordings retain their
own authors' licences and are credited directly below.

## Synthesised sounds

Drift also includes sounds generated in real time by the Web Audio
API with no external source recordings. These are original work and
fall under Drift's MIT licence.

The current synthesised layers are: Birds, Crickets, Void Drone,
Static Field, Warp Drive, and Hypersleep.
